# Gridea Theme 「 Subtle Galaxy 」

> 基于静态博客生成器[Gridea](https://gridea.dev/)的简洁现代风博客网站主题「Subtle Galaxy」

### 安装方法
- 克隆项目到本地
- 复制「 subtle-galaxy 」目录到「 Gridea 」主题文件中
- 重启「 Gridea 」客户端
- 在主题中选择「 Subtle Galaxy 」主题，点击保存
- 预览查看效果
